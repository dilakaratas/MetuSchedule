## METU Scheduler

This project allows Middle East Technical University students to create their course schedules with a friendly user interface.

## Motivation

This project was built with the hopes of making the course selection and schedule planning period easier for METU students.

## Code

This project was built using React.js, Vite, JavaScript, and CSS.

The course data is collected from METU course catalog pages and converted into JSON format to be used in the frontend application.

## Features

- Course schedule visualization
- Department and course listing
- Weekly calendar interface
- JSON-based course data
- User-friendly design

## External Libraries

[React](https://react.dev/)

[Vite](https://vite.dev/)

## Data

The project uses METU course catalog data. The raw course data is processed and cleaned before being displayed in the schedule interface.
